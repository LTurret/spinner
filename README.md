# spinner

Spinner is a little program that trying to handover the loading animation from npm package manager.

```sh
$ python3 spinner.py
╭─┐meow :3┌────────────────────╮
│ ✔ Done!                      │
╰──────────────────────────────╯
╭─┐result┌─────────────────────╮
│ Ur cake is good now meow     │
╰──────────────────────────────╯
```
